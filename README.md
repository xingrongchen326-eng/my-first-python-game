# my-first-python-game
# 🎯 猜數字遊戲

歡迎來到我的第一個 Python 專案！🚀

---

## 📌 專案介紹

這是一個使用 Python 撰寫的簡單猜數字小遊戲。  
電腦會隨機產生一個 1 到 100 之間的整數，  
玩家需要透過輸入數字來猜出正確答案。

---

## 🎮 遊戲玩法

1. 執行程式
2. 輸入 1 到 100 之間的數字
3. 系統會提示：
   - 🔼 太小了
   - 🔽 太大了
4. 持續猜測直到答對為止

---

## 🛠 使用技術

- Python 3
- random 模組（產生隨機數字）

---

## ▶️ 執行方式

請在終端機輸入以下指令：

```bash
python game.py
